# 5-Build

Module 5 Build

The Indie Film Collective website is a multi-page website designed to provide information about the Indie Film Collective and it's associated film festival. It contains a "Welcome" page that briefly summarizes the site, a "Film Screenings" page that lists featured indie films, a "Workshops" page that lists the workshops that are available with descriptions for each, and a "Collaborations" page that briefly describes how the Indie Film Collective website can help people collaborate together.

The HTML is semantic and validated.
The CSS is responsive.
There is no Javascript.
